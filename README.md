# Marks View
Students Results Database Management System

## How to run
1) install nodejs</br>
2) install npm</br>
3) install mongodb</br>
open up your terminal and navigate inside marks-view/marks</br>
run <i>npm install</i></br>
run <i>node app.js</i></br>
</br>
open localhost:3000
