# marks-view
Students Results Database Management System

## how to run
install nodejs</br>
install npm</br>
install mongodb</br>

</br>
navigate inside marks-view/marks</br>
run <i>npm install</i></br>
run <i>node app.js</i></br>
</br>
open localhost:3000
